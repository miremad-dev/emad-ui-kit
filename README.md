# Emad UI Kit

A lightweight UI toolkit for modern web applications.

> 🚧 **Status:** This project is currently under development.

---

## Features

### Current

- ✅ ScrollReveal (v1.0 in progress)

### Planned

- Modal
- Tooltip
- Accordion
- Tabs
- Dropdown
- Toast
- Carousel
- Form validation
- More...

---

## Installation

```bash
npm install emad-ui-kit
```

> Installation will be available after the first npm release.

---

## Quick Start

```javascript
import { ScrollReveal } from "emad-ui-kit";

new ScrollReveal(".section");
```
